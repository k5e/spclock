Speaker's Clock is a countdown app that helps speakers to watch their time.
  • Use the +/- buttons to set the desired minutes. Push longer to advance fast.
  • Push Start to set off. Push Pause to, well, pause.
  • Or just touch the time display for start/stop.
  • Push Reset to set time back to the previously set minutes.
  • Watch the progress bar change color while time runs out.
  • Enjoy!
